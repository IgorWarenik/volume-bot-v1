# Bybit Volume Spike Bot v1 🚀

Асинхронный торговый бот для биржи Bybit (Linear USDT фьючерсы). Бот сканирует все доступные монеты каждую минуту и отправляет уведомление в Telegram, если объем торгов за последнюю минуту аномально превысил среднее значение.

## Особенности
- **Сверхбыстрый сканер:** Использует `aiohttp` и `asyncio` для одновременного опроса более 200 монет за 1-2 секунды.
- **Гибкая настройка:** Изменение множителя объема и таймфреймов прямо на лету через команды в Telegram.
- **Готов к облаку (Replit / Render):** Встроенный модуль `keep_alive` для бесплатного хостинга.

## Установка и запуск

1. Клонируйте репозиторий:
```bash
git clone https://github.com/IgorWarenik/volume-bot-v1.git
cd volume-bot-v1
```

2. Установите зависимости:
```bash
pip install -r requirements.txt
```

3. Настройте конфигурацию:
Создайте файл `config.py` на основе `config.example.py` и впишите туда свои ключи от Telegram бота (получить у @BotFather):
```bash
cp config.example.py config.py
```

4. Запустите бота:
```bash
python3 bot.py
```

## Управление через Telegram
После запуска скрипта перейдите в ваш Telegram-бот и используйте команды:
- `/start` — Запустить сканирование
- `/stop` — Остановить сканирование
- `/status` или `/settings` — Проверить текущий статус и настройки
- `/set_vol 15` — Изменить множитель объема на лету
- `/set_can 30` — Изменить количество анализируемых свечей
- `/set_int 60` — Изменить интервал сканирования в секундах
